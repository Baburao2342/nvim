# nvim

This repository contains **my personal Neovim configuration**.

It is meant to be **simple, beginner-friendly**, and easy to understand. Nothing fancy, no extreme optimizations — just a basic setup that works.

---

## What is this?

This is my Neovim config. I use it for learning, experimenting, and slowly improving my setup as I understand more about Neovim.

If you're also a beginner, feel free to look around and copy parts you like.

---

## Installation

### 1. Backup your existing config (if any)

```bash
mv ~/.config/nvim ~/.config/nvim.backup
```

---

### 2. Clone this repo

```bash
git clone https://github.com/Baburao2342/nvim ~/.config/nvim
```

---

### 3. Start Neovim

```bash
nvim
```

It should automatically install the plugin manager and plugins on first launch.
